Pointcheck.gg (2018)


A friend and I used to sit around and argue over which of us was the better Halo 3 player back in the day. We thought that It would be cool to see the statistic from every match that we had played together, so we made an application that does just that. Fortunately, the creators of Halo kept a match history log for every player who ever played the game. We first asynchronously scrape the match history pages for both players and store each unique match ID into an SQL Server database. We then search for matching IDs to find which matches both players were a part of. The application gathered a minor buzz in the small but passionate Halo 3 community. We had several hundred unique users in the first 6 months but has steadily decreased in traffic since then. Ultimately this was just a passion project that I had a lot of fun making.
